# 课后练习
附件中的工程是一个"Tic-Tac-Toe"游戏。

```
X, select a space
 1  | 2  | 3
--------------
 4  | 5  | 6
--------------
 7  | 8  | 9
```

运行后是这样的一个棋盘。两个玩家依次落子。先排成“横”，“竖”，“斜”一条线的赢。

代码中其他文件忽略，只关注game.rs/board.rs 这两个文件。里面有“TODO”提示。
在提示的地方填充函数内容。
